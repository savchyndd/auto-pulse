# AutoPulse

## Project Overview

This project is for a car rental company. It is built on ReactJS with SASS and uses mockapi.io for interacting with the database.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start

# production mode
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test
```
